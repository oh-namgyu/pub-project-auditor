# pub-project-auditor (한국어 소개)

> 노트북에 쌓인 **모든 Git 저장소를 한 명령으로** Claude Code 로 감사한다. AI 코드 리뷰 + 보안 감사 → 웹 대시보드로 리포트 렌더링.

[English README](README.md)

## 어떤 문제를 푸나

대부분의 개발자에겐 `~/code` 에 반쯤 만든 사이드 프로젝트가 수십 개 쌓여 있습니다. 1년째 안 연 것, 잊어버린 하드코딩 시크릿이 든 것, 나중에 더 깔끔하게 푼 중복 로직으로 가득한 것. 이걸 일일이 다시 열어보는 대신, 디렉토리를 통째로 스캔해 각 저장소에 AI 코드 리뷰와 보안 감사를 돌립니다.

## 핵심 기능

- 로컬 Git 저장소 디렉토리 일괄 스캔
- 각 저장소에 **AI 코드 리뷰 + 보안 감사** ([Claude Code](https://docs.claude.com/en/docs/claude-code/overview) CLI 기반)
- 생성된 Markdown 리포트를 **작은 웹 대시보드**로 렌더링

## 기술 스택

Python 3.9+ · Claude Code CLI · 웹 대시보드 · MIT

## 이 프로젝트의 핵심 포인트

"방치된 사이드 프로젝트 더미"라는 모두가 공감하는 문제를, **AI 에이전트를 배치(batch)로 돌려** 한 명령으로 푸는 자동화입니다. 잊은 시크릿·중복 로직을 한 번에 훑어내는 실용 도구.
